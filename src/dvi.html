SMART CALC

Общее введение

Функционал данного калькулятора маштабен, он работает как стандартный калькулятор.
В случае, если в формуле есть '_x_' он строит его график на заданном диапазоне
Также реализован функционал кредитного и депозитного калькуляторов.

Функционал калькулятора

Вычисление суммы '_+_'
Вычисление разности '_-_'
Вычисление умножения '_*_'
Вычисление деления '_/_'
Вычисление остатка от деления '_mod_'
Вычисление корня '_sqrt_'
Возведение в степень '_^_'
Вычисление логарифма '_log_'
Вычисление натурального логарифма '_ln_'
Вычисление синуса '_sin_'
Вычисление косинуса '_cos_'
Вычисление тангенса '_tan_'
Вычисление арксинус(функция обратная синуса) '_asin_'
Вычисление арккосинуса(функция обратная косинуса) '_acos_'
Вычисление арктангенс(функция обратная тангенса) '_atan_'
Вычисление выражения с X и построение графика'_x_'

Функционал кредитного калькулятора

Вычисляет:
 ⥗  Общей выплаты
 ⥗  Переплаты по кредиту
 ⥗  Ежемесячного платежа
На основе данных о:
 ⥗  Сумме кредита/займа
 ⥗  Процентной ставки
 ⥗  Срока кредита/займа (год|месяц)
 ⥗  Дате выдачи (ГГГГ:ММ:ДД)
 ⥗  Порядке погашения (Аннуитентый|Дифференуированный)

Функционал депозитного калькулятора

Вычисляет:
 ⥗  Сумму на вкладе в конце срока
 ⥗  Начисленные проценты
 ⥗  Сумму налога
На основе данных о:
 ⥗  Сумме вклада
 ⥗  Сроке размещения вклада (год|месяц)
 ⥗  Процентой ставки
 ⥗  Налоговой ставки
 ⥗  Периодичности выплатах
 ⥗  Капитализации процента
 ⥗  Дате открытия вклада
 ⥗  Снитии или пополнении на вклад

Make targets

Таrgets:
 ⥗  all - запуск калькулятора и открытие исполняемого объекта
 ⥗  install - таргет для начала работы с калькулятором
 ⥗  unistall - таргет для удаления программы
 ⥗  clean - очистка "мусорных" файлов
 ⥗  dvi - вывод паспорта проекта
 ⥗  dist - таргет для сборки QT
 ⥗  test - таргет проверки backend-а с помощью unit_test.c
 ⥗  gcov_report - отображение покрытия backend-а unit тестами
